# 前端面试

## js代码面试题

```javascript
let a = {
    b: 3
};

function foo(obj) {
    obj.b = 4;
    return obj;
}

let aa = foo(a);  // 这里a.b赋值为4
console.log(a.b);  // 4
console.log(aa.b);  // 4

aa = {b: 6}
console.log(a.b);  // 4
console.log(aa.b);  // 6
```

```javascript
a = {

        }

a.b.c();
```
